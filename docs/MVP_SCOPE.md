# MVP Scope

## Goal of MVP

Launch a focused, polished website that clearly communicates Uptown Gaming’s brand, venue value, events, and booking path.

---

## In MVP

### Pages
- Home
- Events
- Book the Space
- Venue
- About
- Contact

### Capabilities
- clear navigation
- strong homepage brand presentation
- event discovery / event preview
- venue and amenities presentation
- booking CTA / booking inquiry path
- contact path
- responsive mobile-first layouts
- basic metadata and SEO setup
- basic analytics readiness
- consistent branded UI

### Content
- structured event content
- venue feature content
- booking/CTA content
- community/about copy
- basic FAQ content if useful

---

## Nice to Have in MVP If Lightweight

- FAQ section
- partner/social proof strip
- media/gallery teaser
- small announcement banner
- simple event filtering if easy to support with local content

---

## Out of MVP

- user accounts
- login/auth
- member portal
- organizer dashboard
- database-backed admin tools
- custom booking engine
- custom event management backend
- discount systems
- loyalty systems
- QR/key tag scanning systems
- social scheduling platform
- Twitch/YouTube automation workflows
- large CMS integration unless content maintenance truly requires it

---

## Deferred / Later

### Phase 1.1
- stronger analytics
- expanded event content workflows
- easier content editing
- richer social proof/media content

### Phase 2
- membership concepts
- partner promo concepts
- more advanced event/program support
- more operational tooling if needed

### Future Platform Ideas
- organizer systems
- advanced automation
- discount/community systems
- scan-based operational features

---

## Acceptance Criteria for MVP

The MVP is complete when:

### Brand
- the visual design aligns with the approved brand system
- the site feels like Uptown Gaming and not a generic template
- logo treatment and color use are correct

### Product
- a visitor can understand what Uptown Gaming is within seconds
- a visitor can find events easily
- a visitor can understand the venue offering
- a visitor can reach a booking/contact path easily

### UX
- navigation is clear
- all core pages exist
- the site works well on mobile
- primary CTAs are visible and usable

### Engineering
- code follows the approved architecture
- components are reusable and consistent
- no unnecessary backend or app complexity is introduced
